# 📊 Sales & Demand Forecasting using Machine Learning

## 🔍 Overview
This project focuses on predicting future sales using historical business data.  
It demonstrates how Machine Learning can help businesses make better decisions like inventory planning, staffing, and avoiding losses.

---

## 🎯 Objective
The main goal of this project is to:
- Analyze past sales data  
- Identify trends and patterns  
- Predict future sales using a Machine Learning model  
- Present results in a simple and business-friendly way  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  
- Visual Studio Code  

---

## 📁 Dataset
This project uses a sales dataset (CSV file) containing:
- Order Date  
- Sales  
- Product details (optional)

---

## ⚙️ Project Workflow

### 1. Data Cleaning
- Removed missing values  
- Converted date column into proper format  

### 2. Feature Engineering
- Extracted time-based features like:
  - Month  
  - Day  
  - Year  

### 3. Data Analysis & Visualization
- Daily sales trend  
- Monthly sales trend  
- Rolling average (7-day trend)  

### 4. Model Building
- Used Linear Regression model  
- Trained model using past sales data  

### 5. Forecasting
- Predicted future sales for next 30 days  

### 6. Evaluation
- Used Mean Absolute Error (MAE) to measure accuracy  

---

## 📊 Visualizations
The project includes:
- Monthly Sales Graph  
- Sales Trend with Moving Average  
- Forecast Graph (Actual vs Predicted)  

---

## 📈 Results
- The model successfully identified sales trends over time  
- Forecast shows expected future demand  
- Helps in better business planning
