# 🚀 Machine Learning Internship Projects (Future Interns)

## 📌 Overview

This repository contains all the projects completed as part of the **Machine Learning Internship by Future Interns (2026)**.

The projects focus on solving real-world business problems using Machine Learning and Natural Language Processing (NLP).

---

## 📂 Project Structure

```
Future-Intern-ML-Tasks/
│
├── FUTURE_ML_01  → Sales & Demand Forecasting
├── FUTURE_ML_02  → Support Ticket Classification & Prioritization
├── FUTURE_ML_03  → AI Resume Screening & Candidate Ranking System
```

---

## 🔹 Task 1: Sales & Demand Forecasting

### 🎯 Objective

Predict future sales based on historical data to support business decision-making.

### ⚙️ Features

* Data cleaning and preprocessing
* Time-based feature engineering
* Sales prediction using regression
* Visualization of trends and forecasts

### 🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib

---

## 🔹 Task 2: Support Ticket Classification

### 🎯 Objective

Automatically classify customer support tickets and assign priority levels.

### ⚙️ Features

* Text preprocessing using NLP
* TF-IDF vectorization
* Ticket classification
* Priority prediction (High / Medium / Low)
* Streamlit UI for real-time prediction

### 🛠️ Tech Stack

Python, NLTK, Scikit-learn, Streamlit

---

## 🔹 Task 3: AI Resume Screening System

### 🎯 Objective

Build an intelligent system to match resumes with job descriptions and rank candidates.

### ⚙️ Features

* Resume parsing (PDF/DOCX)
* Semantic similarity using transformer models
* Skill-based matching
* Match score calculation
* Missing skills identification
* Interactive Streamlit UI

### 🛠️ Tech Stack

Python, NLTK, Scikit-learn, Sentence Transformers, Streamlit

## ▶️ How to Run Projects

1. Navigate to any task folder:

```
cd FUTURE_ML_01
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the project:

```
python main.py
```

or (for UI projects):

```
streamlit run app.py
```
